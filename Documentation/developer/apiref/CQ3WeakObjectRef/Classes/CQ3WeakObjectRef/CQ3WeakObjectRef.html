CQ3WeakObjectRef

Wrapper for a zeroing weak reference to a Quesa object, making it easier to use weak references correctly..


When all references to a shareable (reference-counted) object sre disposed, then the object is deleted, and weak references are zeroed. That means that the get() method returns NULL and the isvalid() method returns false.
