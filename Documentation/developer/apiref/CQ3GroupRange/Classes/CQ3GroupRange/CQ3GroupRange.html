CQ3GroupRange

Template used to iterate over a Quesa group using range-based for.


For example, to iterate over geometries in a group, you can do this: 
for (const CQ3ObjectRef& item : CQ3GroupRange<kQ3ShapeTypeGeometry>( group ))
{
  doSomething( item );
}
